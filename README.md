# Rafael Rocha · Portfólio

Portfólio bilíngue de Rafael Rocha, Analista de Dados com atuação em BI, analytics operacional, automação e IA aplicada.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Publicação no GitHub Pages

O projeto inclui o workflow `.github/workflows/deploy-pages.yml`. A cada atualização enviada para a branch `main`, o site estático é gerado e publicado automaticamente.

No repositório do GitHub, mantenha **Settings → Pages → Source** configurado como **GitHub Actions**.

## Idiomas e currículos

- PT-BR é o idioma padrão.
- O seletor PT / EN traduz toda a experiência.
- O botão de currículo acompanha o idioma ativo.
- Tema escuro é o padrão, com alternância para o tema claro e preferência salva no navegador.
